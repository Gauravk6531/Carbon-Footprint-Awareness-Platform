# EcoMind AI Frontend

This is the React + Vite frontend for EcoMind AI, a carbon footprint awareness platform powered by AI.

## Quick Start

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
```

## Features

- 💬 Chat-based natural language interface
- 📊 Carbon footprint calculator
- 🎯 What-if scenario simulator
- 🌱 Responsive, accessible design
- 🔌 Integrates with FastAPI backend

See main README for full documentation.
